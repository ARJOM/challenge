# Front End Challenge

## Como participar
1. Crie um novo fork e branch com seu nome-sobrenome e faça um pull request quando estiver pronto.
2. Envie um email para challenge@somar.io para marcarmos uma conversa.

## Sobre o challenge

É claro que, além de uma boa conversa que queremos ter com você, estamos disponibilizando esse pequeno challenge onde queremos observar um pouco do seu comprometimento técnico, como, boas práticas na implementação, organização do projeto, performance e testes.

O challenge consiste em consumir a nossa API de previsão chamada Nimbus e montar um cartão como o protótipo abaixo.
